Class Overview

Helper classes that facilitates loading keyed batches.

This is useful when trying to load all of foo for a given bar for all bars from disk.

protected abstract Map<K, List<V>> loadBatch (I beginIdForBatch, Integer batchSize)

Loads a batch of entries starting with the given beginIdForBatch. The sum of the sizes of values for entries should at most be batch size. One does not need to worry if the "last" K (bar) according to the ordering is incomplete or not. It will automatically be removed and the nextIdFunction will be applied to the previous element which should result in the last "K" being loaded.
