Class Overview

This class builds modules for a leg of a configurable class. There is a clear conceptual overlap between this class and FactoryModuleBuilder. In the same way that FactoryModuleBuilder offers factories that use runtime args to configure an injectable instance, LegModuleBuilder offers providers that use args available within a configure() method.

Note that in this context, "runtime" means behavior that happens after the application's injector is created.

For the use case when an implementation of an interface requires runtime paramaters in order to get initialized, FactoryModuleBuilder allows the client to bind an interface to a factory, which will accept those runtime parameters and return an instance of the desired interface.

The use case for LegModuleBuilder is similar, however, there is no runtime component. Instead the client is trying to set configurable components of a class to a particular set of Keys.

An example is worth a thousand words. Please have a look at the leg provider wiki page for an illustration.
