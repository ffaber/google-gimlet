public abstract S detectCommonality (Iterable<T> items)

This method introspects each of the given items in order to determine the value of their common feature. It is assumed that:

  1. each item shares a common feature
  2. the iterable is non-empty

If each item is found not to share a common feature (through means of invoking equals(Object) on the results of the accessor method that retrieves the value of the feature), then an CommonalityDetector.InconsistentCommonalityException is thrown. For instance, if each item is purported to have the same value returned from its getValue() method, then this method invokes the method on each, compares the return values, and returns the value if it is equal for all items.

If an empty argument is given, an IllegalArgumentException is thrown.
